## 本地开发 & 阅读

本项目基于 vuepress 进行开发，以提供比 github markdown 更佳的阅读体验

依赖于 `node.js`、`yarn`、`vuepress` 等环境

```sh
# vuepress
yarn global add vuepress

# 本地开发
git clone https://github.com/gdut-yy/The-Mythical-Man-Month-zh.git
cd The-Mythical-Man-Month-zh/
yarn docs:dev
```
